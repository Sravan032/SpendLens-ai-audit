# Reflection

This project helped me understand full-stack application development in a real-world environment.

The biggest learning areas were:

- Frontend and backend integration
- API communication
- OpenAI integration
- Deployment debugging
- CORS configuration
- Persistent database management

One of the biggest challenges was deploying the application successfully on Render while handling database persistence and CORS issues.

I also learned how to structure scalable REST APIs using Spring Boot and build interactive frontend interfaces using Next.js and Tailwind CSS.

This project improved my debugging skills significantly because many deployment and integration issues required careful investigation and troubleshooting.

Overall, this project gave me practical experience building and deploying a complete AI-powered SaaS application.
