# Pricing Data

The application uses predefined pricing assumptions for AI tools and plans.

Example pricing structure:

| Tool | Plan | Estimated Monthly Cost |
|------|------|------------------------|
| ChatGPT | Plus | ₹2000 |
| Claude | Pro | ₹1800 |
| Gemini | Advanced | ₹1900 |

The recommendation engine compares:

- Current plan
- Team size
- Usage patterns
- Estimated savings

The savings logic identifies opportunities where lower-cost plans or consolidated subscriptions can reduce expenses.
